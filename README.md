# VitePress + WhyFrame Demo

A demo of using WhyFrame with VitePress to document a component library.

## Project Setup

```sh
pnpm install
```

## Develop

```sh
pnpm dev
```

## Build

```sh
pnpm build
```

## Serve

```sh
pnpm serve
```
